🐾 Landing Page — ONG Bem-estar Animal

Exercício prático da disciplina de Programação Web.
A página simula o site de uma ONG fictícia de resgate e adoção de animais, contendo as seguintes seções:

Apresentação da ONG
Serviços oferecidos com tabela de preços
Galeria de animais disponíveis para adoção
Loja de produtos
Formulário de contato

Tecnologias utilizadas: HTML5 e CSS3
Observação: projeto fictício para fins educacionais. Os botões de compra e envio de formulário ainda não possuem funcionalidade implementada.
